### 录制规范

详见项目根目录 `SAMPLE_SPEC.md`

### 上传方式

1. **方式一**: 直接上传WAV文件到此目录
2. **方式二**: 打包成ZIP上传，由Agent解压处理

### 技术要求

- 格式: WAV (PCM)
- 采样率: 44100 Hz
- 位深: 16-bit 或 24-bit
- 声道: 单声道
- 时长: 每个音3-5秒

### Agent自动处理

上传采样文件后，Agent会自动：

1. ✅ 验证文件格式和数量
2. ✅ 降噪处理
3. ✅ 音量归一化
4. ✅ 时长截取（0.8秒核心部分）
5. ✅ 加载到音频引擎
